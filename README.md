# JWT를 활용한 REST API 만들기

## 과제 요구사항
- 원하는 주제로 CRUD 구현
- 설계한 ERD 사진 첨부
- Postman 테스트 후 결과 첨부
- 기본적인 [자바 컨벤션](https://newwisdom.tistory.com/m/96)을 준수합니다.
  - intelliJ의 코드 자동 정렬 기능을 사용합니다 (단축키를 찾아봅시다!)
  - 불필요한 개행, 불필요한 공백을 주의합니다.
